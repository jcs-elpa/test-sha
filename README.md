# test-sha
> Test ground for secure hash algorithms.

<p align="center">
  <img src="./etc/demo.gif" width="600" height="140"/>
</p>

This package let you test a list of Cryptographic hash function (CHF) provided
by Emacs' secure hash algorithms.
